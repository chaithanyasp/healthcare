# 🏥 Healthcare Dashboard

A frontend Healthcare SaaS application built using React.  
This app allows users to manage patients, view analytics, and handle authentication.

---

## 🚀 Features

- 🔐 User Authentication (Login / Logout using Firebase)
- 📊 Dashboard Navigation
- 👤 Patient Management (Add & View Patients)
- 📈 Analytics Page (basic insights)
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- React
- TypeScript
- Firebase Authentication
- CSS
